# go-duck-yourself
A Go intro for Rubyists.

![gopher](https://raw.githubusercontent.com/christianpaez/go-duck-yourself/refs/heads/main/gopher.gif)


Topics:
- Basic file server with basic auth
- Benchmarking
- testing
- package management
- concurrency/parallelism
- cli development
- external libs(websocket example)
- mutexes(caching example)
- duck typing vs reflection(JSON formatter example)
- File handling primitives(Ephemeral storage example)
